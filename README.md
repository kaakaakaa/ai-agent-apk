# AI Agent APK

Control your Android phone with AI.

## Features

- Text/Voice control: Open apps, go back, open settings
- Cloud AI: Aliyun Bailian, DeepSeek, Kimi
- Local AI: Gemma 2B, Qwen 0.5B (offline)
- Chat interface
- Stable version

## Quick Start

### Cloud AI (Recommended)

1. Build and install APK
2. Open app and go to Settings
3. Choose AI provider and enter API Key
4. Start chatting

### Local AI (Offline)

Run: scripts/download-models.ps1

## Project Structure

- app/ - Android app source code
- docs/ - User documentation
- scripts/ - Utility scripts

## Build

./gradlew assembleDebug
